# Hello, I'm [Ananta](https://github.com/codeAntu)


A markdown compiler. using react and typescript .

Tokenizes text and builds an Abstract Syntax Tree (AST) through string parsing. The renderer constructs HTML components from the AST recursively using React.

## Lists (beta)
Unordered

- Create a list by starting a line with `+`, `-`, or `*`
- Sub-lists are made by indenting 2 spaces:
- Marker character change forces new list start:
- Ac tristique libero volutpat at

* Facilisis in pretium nisl aliquet

- Nulla volutpat aliquam velit
- Very easy!

Ordered

1. Lorem ipsum dolor sit amet
2. Consectetur adipiscing elit
3. Integer molestie lorem at massa

4. You can use sequential numbers...
5. ...or keep all the numbers as `1.`

Start numbering with offset:

57. foo
1. bar
